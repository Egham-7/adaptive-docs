**Save 60-90% on AI development costs** with intelligent model routing. Works as a drop-in replacement in your favorite VS Code AI assistant.
</Tip>

## Get Your Adaptive API Key

Visit [Sign up here](https://www.llmadaptive.uk/sign-up?redirect_url=/api-platform/orgs) to create an account and generate your API key.

## Quick Setup

<Steps>
  <Step title="Install Cline Extension" icon="download">
    1. Open VS Code Extensions Marketplace
    2. Search for "cline" and install the official extension
    3. Click the Cline icon in the sidebar
  </Step>
  <Step title="Configure API Settings" icon="gear">
    Select **"Use your own API Key"** and enter:
    ```text
    API Provider: OpenAI Compatible
    Base URL: https://api.llmadaptive.uk/v1
    API Key: your-adaptive-api-key
    Model: [leave empty for intelligent routing]
    ```
  </Step>
  <Step title="Start Coding" icon="play">
    Begin using Cline for code generation, debugging, and AI-assisted development with cost-optimized routing.
  </Step>
</Steps>

## Alternative Setup Methods

<CodeGroup>
```json Settings File
// In VS Code settings.json
{
  "cline.provider": "openai-compatible",
  "cline.baseUrl": "https://api.llmadaptive.uk/v1",
  "cline.apiKey": "your-adaptive-api-key"
}
```

```bash Environment Variables
export OPENAI_BASE_URL=https://api.llmadaptive.uk/v1
export OPENAI_API_KEY=your-adaptive-api-key
```

```text Manual Configuration
1. Open Cline settings in VS Code
2. Set API Provider to "OpenAI Compatible"
3. Enter Base URL and API key as shown above
```
</CodeGroup>

## Troubleshooting

<AccordionGroup>
<Accordion title="Connection Issues" icon="wifi-off">
- Verify your API key at [llmadaptive.uk/dashboard](https://www.llmadaptive.uk/dashboard)
- Check that Base URL is set to `https://api.llmadaptive.uk/v1`
- Restart VS Code after configuration changes
</Accordion>

<Accordion title="Model Selection" icon="robot">
- Leave model field empty for intelligent routing (recommended)
- Specific models: `claude-3-5-sonnet-20241022`, `gpt-4o`, `gpt-4o-mini`
- Test connection with a simple request after setup
</Accordion>
</AccordionGroup>
