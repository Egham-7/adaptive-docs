Anthropic Claude-compatible API with automatic model selection.

## Quick Start

<CodeGroup>
```bash cURL
curl https://api.llmadaptive.uk/v1/messages \
  -H "Authorization: Bearer apk_123456" \
  -H "anthropic-version: 2023-06-01" \
  -d '{
    "model": "adaptive/auto",
    "max_tokens": 1000,
    "messages": [{"role": "user", "content": "Hello!"}]
  }'
```
